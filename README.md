## About The Project

This portfolio is designed to support resume sharing with a clean single-page layout. It highlights:

- professional introduction
- frontend skills and tools
- selected React-based projects
- career highlights
- contact details for opportunities

The UI follows an editorial, resume-friendly style with strong typography, structured sections, and a clean black-and-white card layout.

## Featured Projects

### Netflix Clone

- Built with React.js, Vite, Firebase, and OMDB API
- Includes authentication and dynamic movie data rendering
- Responsive layout for desktop, tablet, and mobile

### Zoobiya E-Commerce Frontend

- Built with React.js and Vite
- Includes product listing, category filtering, search, and cart UI
- Uses Material UI and Bootstrap for interface components

### University Landing Page

- Built as a responsive single-page React website
- Includes hero, courses, testimonials, and contact sections
- Focused on reusable components and clean layout structure

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
